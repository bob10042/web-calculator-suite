About Advanced Web Calculator

A feature-rich web calculator built with modern web technologies.

Features:

  • Basic arithmetic operations (+, -, ×, ÷)
  • Scientific functions (sin, cos, tan, √)
  • Electrical engineering (3-phase power, impedance)
  • Physics constants and calculations
  • Programmable console with variable storage
  • Memory operations (MC, MR, MS, M+)
  • Calculation history with export
  • Dark/Light theme toggle
  • Responsive design for all devices
  • Keyboard support

Version: 1.0.0

Technologies: HTML5, CSS3, JavaScript ES6+
